## 4. 주요 기능

- 명함 스캔 및 자동 정보 추출
- AI 기술을 활용한 정확한 명함 정보 인식
- 수동 입력 및 수정 기능 제공
- 지도 기반 명함 관리
- 명함 소유자의 회사 위치를 지도에서 한눈에 확인
- 주변 비즈니스 파트너 탐색 용이
- 체계적인 인맥 관리
- 명함별 상담 기록 관리
- 중요 고객 생일 알림
- 맞춤형 메모 기능
- 캘린더 일정 관리
- 비즈니스 미팅 및 일정 관리
- 고객별 일정 체계적 관리
- 편리한 연락 기능
- 전화, 문자 바로 연결
- 주소 정보 기반 길찾기 지원
